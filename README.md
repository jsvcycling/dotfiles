# Dotfiles

My standard configurations for Linux.

Inspiration has been taken from (in alphabetical order):

* [jessfraz/dotfiles](https://github.com/jessfraz/dotfiles)
* [jonhoo/configs](https://github.com/jonhoo/configs)
* [ohmyzsh/ohmyzsh](https://github.com/ohmyzsh/ohmyzsh)

## Installation

When installing, any dot-files in your home directory will be overwritten with
symlinks to wherever you've cloned this repository. Make sure to backup your
existing dot-files first.

```sh
$ make
```
